## Prerequisite

Make sure you have the necessary dependencies:

```
bash
npm install
```

## Running app

Starts the service on a port specified through the environment variable, otherwise default is port 3000:

```
bash
npm start
```

## Running tests

- All tests
```
bash
npm test
```

- Unit tests
```
bash
npm run test:unit
```

- Integration tests
```
bash
npm run test:integration
```

- Lint
```
bash
npm run lint
```
